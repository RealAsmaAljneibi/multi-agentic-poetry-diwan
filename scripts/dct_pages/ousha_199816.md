قصيدة الهدية: الفائزة بجائزة الشيخ محمد بن راشد آل مكتوم

| الهــديه يتنــي اصـدافي | هـوب لـي مـن أول آحيده |
| --- | --- |
| يـا زعيـم الفكر يالوافي | فيـك معنى العرف تفريده |
| والفهـم مـن نبعك الصافي | والأدب فنـــه وتجويــده |
| همـت بيـن اعصـار عسـافي | فـي شـموخ اجبال امشيده |
| لا ذرى عنــدي ولا اتقـافي | في النزول أو عسر تصعيده |
| بيـن مـوي ابحـور ملقافي | مايصــك القــوع بليـده |
| ذاك لــي خلانــي امجـافي | فـي احتراس وفكري آعيده |
| مـا نولفهـا مـن الطـافي | لـي على المزبون تقليده |
| مـن نضـار أو تـبر شفافي | كـم خشـف زان بـه جيـده |
| تهـت بيـن اشـعار عرافـي | فـي بيـان السبت تحديده |
| كــل منهـم خـايعه عـافي | مـا يـدوس الرسم تسنيده |
| يـا محـبي عـوج لا نجـافي | وانتهــل صـافي مواريـد |
| إن صـفا لـك منهـل صـافي | مـن منالـك فـي تود يده |
| عـود لـه والعـود لانصافي | والحقـه لو داره ابعيده |
| لاتشــيل همـوم واحسـافي | كلهـا مـن غيـرك ازهيده |
| يا هوى الخفرات يا سنافي | اشـقرٍ ينيـب عـن الصيده |
| ويـن شـيل الشربت إلحافي | والرهام أو ناعم الميده |
